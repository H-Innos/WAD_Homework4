# WAD_Homework4

## Backend
Navigate to the backend directory <br>
```
cd backend
```
Install the dependencies<br>
```
npm install
```
Start the server <br>
```
npm run start
```
## Frontend 
Navigate to the frontend directory <br>
```
cd frontend
```
Install the dependencies <br>
```
npm install
```
Run the project 
```
npm run serve
```
